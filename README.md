This project is a Deep Learning application that recognizes handwritten digits (0-9) in real-time. It uses a Convolutional Neural Network (CNN) trained on the famous MNIST dataset and features a graphical user interface (GUI) built with Tkinter.
🚀 Features

    Real-time Recognition: Draw any digit on the canvas and get instant predictions.

    CNN Architecture: Built using TensorFlow and Keras with Convolutional and Pooling layers for high accuracy (~99%).

    Interactive UI: A simple and clean drawing board made with Python's Tkinter library.

    Auto-Save: The model is saved automatically as digit_recognizer_model.h5 after training, so you don't have to train it every time.
